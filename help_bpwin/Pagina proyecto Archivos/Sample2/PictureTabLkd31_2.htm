Name 
Mostrar mapa
Number A33
Definition 
El usuario podrá ver el mapa con las ubicaciones de las entidades registradas en la base de datos, donde pondrá su posición actual para identificar la distancia entre él y las entidades.
 

Input Arrow
Name Definition 
coordenadas usuario
 
Son las coordenadas actuales donde se presenta el usuario, también pueden ser asignadas por el mismo manualmente moviendo el marcador de usuario.
 
Enviar posición de entidades
 
De la base de datos, se saca la latitud y longitud de cada marcador de entidad registrado para poder mostrar en el mapa.
 
Ingresar como usuario
 
Se ingresa a la página principal, Para los usuarios comunes, la aplicación es abierta, con casi ningún privilegio precisamente por ser sólo un usuario.
 
Output Arrow
Name Definition 
selesccionar entidad
 
Se seleciona una entidad cualesquiera, no solo para recabar información de la misma, sino también ver si está activa o inactiva.
