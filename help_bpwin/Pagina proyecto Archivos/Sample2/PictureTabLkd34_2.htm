Name 
Análisis
Number A1
Definition 
El análisis del proyecto es la parte donde se evalua el proceso del negocio, mediante encuestas y entrevistas.
 

Input Arrow
Name Definition 
coordenadas usuario
 
Son las coordenadas actuales donde se presenta el usuario, también pueden ser asignadas por el mismo manualmente moviendo el marcador de usuario.
 
Direcciones de ATM
 
los datos de latitud y longitud de los ATM's.
 
Direcciones de sucursales
 
Latitud y longitud de las sucursales.
Mapas 
La entrada de datos del mapa, debido a que se manejará el Api de Google Maps, simplemente se convierte en una llamada de una función.
Metadatos 
Información extra de las calles, posicionamiento, direcciones, fotos,información de sucursales y otros.
 
Output Arrow
Name Definition 
Información detallada del análisis
 
Se tiene la información necesaria después de realizar las respectivas encuestas y entrevistas, de tal manera que se tenga una mejor idea de la problematica y como poder solucionarlo.
 
Mechanism Arrow
Name Definition
manual 
Toda operación que se raliza de forma manual.
