Name 
SIG BASADO EN TECNOLOGÍA WEB ORIENTADO A LA  BUSQUEDA DE RUTAS ENTRE  SUCURSALES, ATM’S Y USUARIO
Number A0
Definition 
Es el desarrollo de un sistema GIS en un ambiente web, donde para la optimización de una ruta entre sucursales, ATM's y el usuario


Input Arrow
Name Definition 
coordenadas usuario
 
Son las coordenadas actuales donde se presenta el usuario, también pueden ser asignadas por el mismo manualmente moviendo el marcador de usuario.
 
Direcciones de ATM
 
los datos de latitud y longitud de los ATM's.
 
Direcciones de sucursales
 
Latitud y longitud de las sucursales.
Mapas 
La entrada de datos del mapa, debido a que se manejará el Api de Google Maps, simplemente se convierte en una llamada de una función.
Metadatos 
Información extra de las calles, posicionamiento, direcciones, fotos,información de sucursales y otros.
 
Output Arrow
Name Definition
Conclusiones 
Es la información recabada, después de validar el prototipo junto con algunas recomendaciones.
Prototipo 
El desarrollo del prototipo mas adecuado para las respectivas validaciones.
Reportes 
Son informes que nos muestran como va el progreso del sistema y asi también tener un control del mismo.
